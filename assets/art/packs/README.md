# 서드파티 에셋 팩

이 폴더의 에셋은 **저장소에 커밋하지 않습니다**(`.gitignore` 처리).
대부분의 픽셀아트 팩 라이선스에 *"에셋 자체의 재배포 금지"* 조항이 있어,
공개 저장소에 넣으면 애매해지기 때문입니다. 프로젝트에 **사용**하는 것은 허용됩니다.

새로 클론했다면 아래 목록대로 직접 받아서 이 폴더에 넣으세요.

## 필요한 팩

| 폴더 | 출처 | 가격 | 라이선스 요약 |
|---|---|---|---|
| `knight/` | [FREE - Knight 2D Pixel Art](https://xzany.itch.io/free-knight-2d-pixel-art) — Mattz Art | 무료 (name your own price) | 상업적 사용·수정 가능 / 크레딧 불필요 / **에셋 재판매·재배포 금지** / NFT 금지 |

## 받는 방법

1. 위 링크 접속 → `Download Now`
2. 금액란을 비우고 **`No thanks, just take me to the downloads`** 클릭
3. zip을 받아 이 폴더에 압축 해제 (예: `assets/art/packs/knight/`)
4. Godot 임포트:
   ```
   "/c/Program Files (x86)/Godot/Godot_console.exe" --headless --path . --import
   ```

## 주의

- 팩을 추가하면 **`CREDITS.md`에도 출처·라이선스를 기록**할 것
- 일부 팩은 **"AI 학습·생성에 사용 금지"** 조항이 있다(예: aamatniekss의 Fantasy Knight).
  해당 팩의 스프라이트를 이미지 생성 AI에 입력하지 말 것.
  Mattz Art 팩에는 이 조항이 없다.
